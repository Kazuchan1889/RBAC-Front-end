# RBAC Frontend

> 🔐 Role-Based Access Control System — Angular 17 Frontend

## Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Run development server
npm start
# → Opens at http://localhost:4200
```

## Build

```bash
# Production build
npm run build

# Output → dist/rbac-frontend/
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm start` | Run dev server (port 4200) |
| `npm run dev` | Run dev server + open browser |
| `npm run build` | Production build |
| `npm run watch` | Build with file watching |

## Tech Stack

- **Angular 17** (Standalone Components)
- **TypeScript 5.4**
- **CSS** (Custom Design System with CSS Variables)
- **Canvas API** (Particle Animations)
- **Angular Animations** (Entry/Exit transitions)

## Features

- ✨ Modern dark UI with glassmorphism
- 🎨 Particle canvas background animation
- 🔄 Animated RBAC logo with orbiting dots
- 📝 Reactive form with real-time validation
- 🔐 Show/hide password toggle
- ⚡ Loading & success states
- 🌐 SSO buttons (Microsoft, Google)
- 📱 Fully responsive
